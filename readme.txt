== Description ==

BuddyPress Links is a drop-in link, image, video and other rich media sharing component for BuddyPress 2.0 and higher. This plugin allows you to reward users with points for creating links or voting on them.

= myCred for BuddyPress Links Features =
Here are the most notable features of myCred for BuddyPress Links plugin.

* Awarding points for adding links.
* Awarding points for voting on links.
* Badge support.
* Translation ready.

= Plugin Requirements =

* [myCred 1.8+](https://wordpress.org/plugins/mycred/)
* [BuddyPress Links](https://wordpress.org/plugins/buddypress-links/)
* WordPress 5.0+
* PHP 5.3+

== Installation ==

1. Go to Plugins > Add New.
2. Under Search, type myCred for BuddyPress Links
3. Find myCred for BuddyPress Links and click Install Now to install it
4. If successful, click Activate Plugin to activate it and you are ready to go.
